# نظام الوقار — إدارة برنامج حفظ القرآن الكريم

نظام إدارة برنامج الوقار لتحفيظ القرآن الكريم (قسم البنات) — الدورة الرابعة عشرة، جمعية تعلّم للقرآن.

**الموقع الحي:** https://alwaqar.org

## التقنيات

- **الواجهة:** React 18 + TypeScript + Vite + shadcn/ui + Tailwind CSS
- **التوجيه:** React Router (SPA)
- **جلب البيانات:** TanStack React Query
- **قاعدة البيانات والمصادقة:** Supabase (Postgres + Auth + RLS)
- **الاستضافة:** Vercel — مربوط بمستودع GitHub، أي push على فرع `main` ينشر تلقائيًا للإنتاج

## الصفحات الرئيسية

- **صفحات عامة (بدون تسجيل دخول):** التعهد `/pledge`، المقابلة `/interview`، السداد `/payment` — تبحث برقم الهوية الوطنية
- **صفحات إدارية (تتطلب تسجيل دخول):** لوحة التحكم، المتقدمات، الطالبات، المعلمات، الحلقات، الحضور، التسميع، الاختبارات، المقابلات، التعهدات، الاستئذانات، المخالفات، المستخدمون، الإعدادات

## التشغيل محليًا

```sh
npm install
npm run dev        # خادم التطوير على المنفذ 8080
```

## الأوامر

```sh
npm run build      # بناء نسخة الإنتاج
npm run lint       # فحص الكود
npm run test       # تشغيل الاختبارات (vitest)
```

## النشر

```sh
git push           # ينشر تلقائيًا على alwaqar.org عبر Vercel
```

## ملاحظات مهمة

- ملف `vercel.json` ضروري — يحتوي rewrites توجّه كل المسارات إلى `/index.html` حتى يعمل React Router. بدونه ترجع المسارات المباشرة (مثل `/students`) خطأ 404.
- ترحيلات قاعدة البيانات ملفات SQL مرقّمة (`01_…`, `02_…`) خارج هذا المجلد، وتُنفَّذ يدويًا في Supabase SQL Editor.
- الواجهة بالعربية مع دعم RTL كامل.
